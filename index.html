    Student Development Kit
    Student Development Kit is the official android application of the Codigo Plataforma, coding club of Silicon Institute of Technology, Sambalpur. It's a fully-featured application where users can sign in to avail benefits like real-time questions which are updated on weekly basis. It also features several sections for practicing cutting-edge technologies and job-ready skills like Mobile Application Development, Web Development, Competitive Programming, etc. Get updated with the announcements and latest trends in programming through the application.
